On this page

最新 Vue3 + TS 高仿 ElementPlus 打造自己的组件库 ​

安装 ​

bash
npm i @create-element --save

开始使用 ​

全局使用

js
// 引入所有组件
import CtElement from 'create-element'
// 引入样式
import 'create-element/dist/index.css'

import App from './App.vue'
// 全局使用
createApp(App).use(CtElement).mount('#app')
vue
<template>
	<ct-button>我是 Button</ct-button>
</template>

单个导入

Create-Element 提供了基于 ES Module 的开箱即用的 Tree Shaking 功能。

vue
<template>
	<ct-button>我是 Button</ct-button>
</template>
<script>
import { CtButton } from 'create-element'
export default {
	components: { CtButton },
}
</script>
